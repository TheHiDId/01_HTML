p(Paragraph, 문단, 단락) 태그

글을 작성하는 용도의 태그

pre 태그처럼 형식이 유지되지는 않음

글의 주제가 바뀔 때 마다 p태그를 새로 작성

문단 사이에 여백이 생생 됨

글자에 적용되는 태그

b(Bold) 태그 : 단순히 텍스트를 굵게 표시하는 태그
i(Italic) 태그 : 단순히 텍스트를 기울여 표시하는 태그
strong 태그 : 텍스트를 강조하는 태그
em 태그 : 문장의 맥락을 변화시키는 태그(강조) 
      * 웹 접근성이란?
      - 일반인 뿐만 아니라 장애인, 고령자 등을 위해 웹사이트에서 제공하는 모든 서비스를 이용할 수 있도록 보장하는 것(웹 개발 의무 사항)  
    
      mark 태그 : 형광펜 효과를 주는 태그
      u(under) 태그 : 밑줄을 긋는 태그
      s 태그 : 가운데 선을 긋는 태그(취소, 오타)
      del 태그 : 삭제 선을 긋는 태그(있다가 없어짐)
      *s, del 태그 용도에 맞게 사용!
      걸그룹 멤버 명단 : 미영(탈퇴), 숙희, 영희영숙
      small 태그 : 텍스트를 작게 출력하는 태그
      sup 태그(윗첨자) : x3
      sub 태그(아랫첨자) : log2
      abbr(Abbreviation, 약어) 태그: 커서를 올리면 툴팁 제공
      HTTP

글자 관련 태그 연습

태그들은 해당 태그 내에서 중첩으로 사용가능하다.
굵은 글자를 사용할 수도 있고, 기울이거나 취소선을 넣는 등
다양하게 사용할 수 있다.
