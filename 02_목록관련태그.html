ol(Ordered List) 태그 : 순서가 있는 목록

  1. 김밥
  2. 덮밥
  3. 햄버거
    * type 속성 : 표시되는 숫자 형식 지정
    type="a" : 영어 소문자
    type="A" : 영어 대문자
    type="i" : 로마 숫자 소문자
    type="I" : 로마 숫자 대문자
    type="1" : 숫자, 기본값

    * start 속성 : 해당 목록이 몇번부터 시작하는지 지정

    * reversed 속성 : 순서 반대로(역순)

Emmet : HTML, CSS 빠른 자동완성 지원 플러그인

    태그명 : 태그 시작 및 종료 태그 자동 완성
    > : 태그 내부에 작성(하위 생성)
    + : 태그 다음에 작성(형제)
    *숫자 : 숫자만큼 생성
    {} : 태그 내부에 작성될 내용
    $ : 1부터 1씩 증가하는 숫자(시퀀스)
