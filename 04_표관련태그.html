table 관련 태그

  1. table(표) 태그

            - 웹 문서에서 자료를 정리하기 위해 사용하는 태그
        - table 태그는 행, 열을 작성하기 위한 영역을 지정하는 태그

  2. tr(Table Row) 태그

            - 테이블의 한 행을 나타내는 태그

  3. td(Table Data) 태그

            - 테이블에 출력될 값을 작성하는 태그
        - 각 행의 열(column)의 역할 == Excel의 한 Cell과 동일

  4. th(Table Header) 태그

            - 열의 제목을 나타내는 태그
        - td와 같은 레벨로 작성(td를 th로 대체가 가능)
        - 굵은 글씨 + 가운데 정렬

  5. caption 태그

            - 표 또는 이미지에 설명을 작성하는 태그

  6. thead 태그

            - 테이블 상단 영역(컬럼 제목 영역)

  7. tbody 태그

            - 테이블 중단 영역(값 작성 영역)

  8. tfoot 태그

            - 테이블 하단 영역(합계 작성 영역)

  9. colspan(열 병합) 속성

            - 열로 나열된 가로 방향으로 병합

  10. rowspan(행 병합) 속성

            - 행으로 나열된 세로 방향으로 병합
